Identidad visual completa para una marca de comidas rápidas

EL PROYECTO

Una marca que
sabe a fuego.

Fueggo necesitaba una identidad que comunicara intensidad, rapidez y sabor desde el primer vistazo. El reto fue construir una marca que funcionara tanto en un uniforme como en una caja de empaque — consistente, agresiva y memorable.

El resultado: un sistema visual completo con logo, paleta, tipografía y aplicaciones reales listas para producción.
